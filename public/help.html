StudentsDB

Easily filter the Student Database :D

Help

StudentsDB is a simple web application that allows you to filter the student database of Parramatta High School. You can filter the students based on their name, student reference number (SRN), roll class, gender, and house.

Simply enter the search query in the search bar and select the desired filter options. The students that match the search query and filter options will be displayed below in the results section.

For any queries or feedback, please contact the developer at adil.sheeth@education.nsw.gov.au.
